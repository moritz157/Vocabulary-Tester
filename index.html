What is my Vocabulary Tester?

In general, my Vocabulary Tester is a pretty simple program to practice for classtests. But can test not only vocabulary: You can use my program to test anything what you want, for Example

  • Captial cities (Geography)
  • Year dates (History) and unlimited more

Techical information

This application is programmed in Delphi Language which is almost similar to Pascal. It definitely works on Windows 7 but I didn't tried it on other Windows yet. The '.vok'-Files are lists with the questions in the first half and the appropriate answers are in the second half.
